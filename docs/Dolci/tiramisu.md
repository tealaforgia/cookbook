# Tiramisù

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `300g [≈6]` | Uova |
| :fontawesome-regular-square: `500g` | Mascarpone |
| :fontawesome-regular-square: `100g` | Zucchero |
| :fontawesome-regular-square: `10g` | Succo di limone |
| :fontawesome-regular-square: `≈300g` | Savoiardi |
| :fontawesome-regular-square: `≈500g` | Caffè[^1] |
| :fontawesome-regular-square: `250g` | <span class="opzionale">Panna</span> |
| :fontawesome-regular-square: `1 baccello` | <span class="opzionale">Semi di vaniglia</span>[^2] |
| :fontawesome-regular-square: | Cacao amaro in polvere |

---

## :lucide-chef-hat: Preparazione

- [ ] Preparare il caffè e lasciarlo a raffreddare.
- [ ] Montare a neve fermissima l'albume con il succo di limone e `1/3` dello zucchero.
- [ ] Montare i tuorli con `2/3` dello zucchero, quindi aggiungere il mascarpone freddissimo (eliminando il siero se presente) e montare la crema.
- [ ] Incorporare delicatamente l'albume alla crema con la frusta.
- [ ] Alternare uno strato di crema con uno di savoiardi inzuppati nel caffè, chiudendo con la crema.
- [ ] Lasciare riposare in frigo per `≥12h`.
- [ ] <span class="opzionale">Montare la panna freddissima aggiungendo i semi di vaniglia con la frusta e ricoprire il tiramisù con la panna montata usando una sac a poche.</span>
- [ ] Spolverare il tiramisù con il cacao.

[^1]: Caffè decaffeinato
[^2]: Essenza di vaniglia
