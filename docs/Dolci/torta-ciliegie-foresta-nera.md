# Torta di ciliegie della Foresta Nera [Schwarzwälder Kirschtorte]

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `1` | [Pan di Spagna al cioccolato](../Preparazioni-di-base/pan-di-spagna-cioccolato/#ingredienti){ data-preview } |
| :fontawesome-regular-square: `1kg` | [Crema Chantilly](../Preparazioni-di-base/crema-chantilly/#ingredienti){ data-preview } |
| :fontawesome-regular-square: `1 bicchierino` | Kirsch [^1] |
| :fontawesome-regular-square: `1kg` | Ciliegie denocciolate |
| :fontawesome-regular-square: `100g` | Zucchero |
| :fontawesome-regular-square: | Scaglie di cioccolato |

---

## :lucide-chef-hat: Preparazione

- [ ] Mettere il [pan di Spagna al cioccolato](../Preparazioni-di-base/pan-di-spagna-cioccolato/#preparazione){ data-preview } nel frigo.
- [ ] Fare una riduzione delle ciliegie con lo zucchero a fuoco medio per `≈30min`. Separare le ciliegie dalla riduzione.
- [ ] Quando la riduzione si sarà raffreddata aggiungere il Kirsch e mettere in frigo sia la bagna che le ciliegie.
- [ ] Quando tutti gli ingredienti si saranno raffreddati nel frigo tagliare il pan di Spagna in 3, quindi spennellare la bagna e farcire con le ciliegie e la [crema chantilly](../Preparazioni-di-base/crema-chantilly/#preparazione){ data-preview }, lasciando poco più di `1/3` della crema per ricoprire la torta.
- [ ] Decorare a piaciere con la crema Chantilly, le scaglie di cioccolato e ciliegie fresche.
