# Torta di mandorle

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `75g` | Zucchero |
| :fontawesome-regular-square: `170g + stampo` | Burro |
| :fontawesome-regular-square: `200g [≈4]` | Uovo |
| :fontawesome-regular-square: `80g + stampo` | Farina |
| :fontawesome-regular-square: `80g` | Amido di mais[^1] |
| :fontawesome-regular-square: `125g` | Mandorle |
| :fontawesome-regular-square: `1 bustina` | Lievito per dolci |
| :fontawesome-regular-square: `1 pizzico` | Sale |

???- example "Proporzioni"

    **Uovo** (`g`) = `n`

    | Quantità | Ingrediente |
    | :-- | :-- |
    | `n×15÷44` | Zucchero |
    | `n×17÷22` | Burro |
    | `n×4÷11` | Farina |
    | `n×4÷11` | Amido di mais |
    | `n×25÷44` | Mandorle |

---

## :lucide-chef-hat: Preparazione

- [ ] Frullare lo zucchero e le mandorle.
- [ ] Montare con la frusta le uova e il miscuglio di zucchero e mandorle, quindi aggiungere il burro ammorbidito.
- [ ] Sostituire la frusta con la frusta gommata e aggiungere la farina e la maizena setacciate, il lievito e il sale, infine amalgamare.
- [ ] Versare in una teglia imburrata e infarinata.

[^1]: Fecola di patate; Amido di frumento

## :lucide-heater: Cottura

Cuocere in forno non preriscaldato non ventilato a `170°C` per `35min`.
