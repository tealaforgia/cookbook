# Limoncello

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: | Limoni[^1] |
| :fontawesome-regular-square: | Alcol etilico |
| :fontawesome-regular-square: | Acqua |
| :fontawesome-regular-square: | Zucchero |

???+ example "Proporzioni [^2]"

    **Buccia di limone** (`g`) = `n`

    | Quantità | Ingrediente |
    | :-- | :-- |
    | `n×5÷2` | Alcol etilico |
    | `m×5÷4` | Zucchero |
    | `m×5÷2` | Acqua |

---

## :lucide-chef-hat: Preparazione

- [ ] Pelare i limoni con un pelapatate per avere solo la buccia più esterna e non la parte bianca.
- [ ] Tagliare le scorze in piccoli pezzi e lasciare macerare con l'alcol in un luogo buio per `1-3 giorni`[^3].
- [ ] Filtrare le bucce e aggiungere lo zucchero.
- [ ] Aggiungere l'acqua, usandola anche per sciacquare le bucce, e lasciare riposare per `≥1 settimana`.

[^1]: Preferibilmente abbastanza verdi, con buccia edibile
[^2]: Tutti gli ingredienti sono misurati in `g`, e non in `mL`
[^3]: Per avere un limoncello più aromatico `1 giorno`, per avere un limoncello più intenso `3 giorni`
