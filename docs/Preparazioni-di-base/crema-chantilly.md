# Crema Chantilly

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `1000g` | Panna fresca |
| :fontawesome-regular-square: `100g` | Zucchero a velo |
| :fontawesome-regular-square: `1 baccello` | Semi di vaniglia[^1] |

???- example "Proporzioni"

    **Panna** (`kg`) = `n`

    | Quantità | Ingrediente |
    | :-- | :-- |
    | `n÷10` | Zucchero a velo |

---

## :lucide-chef-hat: Preparazione

- [ ] Far raffreddare la panna nel freezer (senza farla congelare) e il contenitore dove verrà montata nel frigo.
- [ ] Montare la panna freddissima insieme allo zucchero a velo e i semi della bacca di vaniglia.

[^1]: Essenza di vaniglia
