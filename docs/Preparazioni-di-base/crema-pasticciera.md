# Crema pasticciera

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `80g [≈5]` | Tuorlo d'uovo |
| :fontawesome-regular-square: `80g` | Zucchero |
| :fontawesome-regular-square: `30g` | Amido di riso[^1] |
| :fontawesome-regular-square: `400g` | Latte |
| :fontawesome-regular-square: `1` | Baccello di vaniglia[^2] |
| :fontawesome-regular-square: `1 limone` | Scorza di limone |
| :fontawesome-regular-square: `0 - 1/2 limone` | <span class="opzionale">Scorza di limone grattugiata</span> |
| :fontawesome-regular-square: `0 - 15g` | <span class="opzionale">Gelatina alimentare</span> |

???- example "Proporzioni"

    **Latte** (`g`) = `n`; **Burro** (`g`) = `m`

    | Quantità | Ingrediente |
    | :-- | :-- |
    | `n` | Zucchero |
    | `n×3÷8` | Amido di riso |
    | `n×5` | Latte |
    | `n÷80` | Baccello di vaniglia |
    | `n÷80` | Scorza di limone |
    | `0 - n÷6` | Gelatina alimentare |

---

## :lucide-chef-hat: Preparazione

- [ ] Mettere nel freezer una ciotola.
- [ ] Portare a bollore il latte in un pentolino con la scorza di limone<span class="opzionale">, la scorza di limone grattugiata</span> e la vaniglia.
- [ ] <span class="opzionale">Mettere la gelatina a bagno</span>.
- [ ] Amalgamare con una frusta i tuorli con lo zucchero, quindi aggiungere la maizena o l'amido di riso e amalgamare.
- [ ] Aggiungere il latte bollente poco a poco sbattendo, <span class="opzionale">aggiungere la gelatina</span>, quindi riscaldare sul fuoco e sbattere con la frusta fino a che la crema non si addensa.
- [ ] Versare velocemente la crema pasticciera nella ciotola fredda e sbattere con la frusta per `3min`.
- [ ] Coprire con pellicola e far raffreddare la crema.

[^1]: Amido di mais  
[^2]: Semi di vaniglia; Essenza di vaniglia
