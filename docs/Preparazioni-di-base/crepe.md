# Crêpe

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `165g [≈3]` | Uovo |
| :fontawesome-regular-square: `250g` | Farina |
| :fontawesome-regular-square: `500g` | Latte |
| :fontawesome-regular-square: `1 noce` | Burro |
| :fontawesome-regular-square: `1 pizzico` | Sale |

???- example "Proporzioni"

    **Uovo** (`g`) = `n`

    | Quantità | Ingrediente |
    | :-- | :-- |
    | `n×50÷33` | Farina |
    | `n×100÷33` | Latte |

---

## :lucide-chef-hat: Preparazione

- [ ] Sbattere le uova con la farina, aggiungendo lentamente il latte ed aggiungere un pizzico di sale.
- [ ] Far sciogliere una noce abbondante di burro nella padella e versare tutto l'eccesso di burro nell'impasto.

## :lucide-flame: Cottura

Cuocere nella padella a fuoco medio-alto.
