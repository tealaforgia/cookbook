# Pan di Spagna al cioccolato

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `300g [≈6]` | Uovo |
| :fontawesome-regular-square: `90g + stampo` | Farina |
| :fontawesome-regular-square: `50g` | Cacao in polvere |
| :fontawesome-regular-square: `150g` | Zucchero |
| :fontawesome-regular-square: `1/2 limone` | Scorza di limone grattugiata |
| :fontawesome-regular-square: `1 cucchiaino` | Essenza di vaniglia |
| :fontawesome-regular-square: `1 pizzico` | Sale |
| :fontawesome-regular-square: `stampo` | Burro |

???- example "Proporzioni"

    **Uovo** (`g`) = `n`

    | Quantità | Ingrediente |
    | :-- | :-- |
    | `n×3÷10` | Farina |
    | `n÷6` | Cacao in polvere |
    | `n÷2` | Zucchero |

---

## :lucide-chef-hat: Preparazione

- [ ] Setacciare insieme la farina e il cacao per miscelarle.
- [ ] Montare alla massima velocità le uova intere con lo zucchero, l'essenza di vaniglia, la scorza di limone e un pizzico di sale per `20min`.
- [ ] Aggiungere `1/2` del miscuglio di farina e cioccolato e mescolare molto delicatamente con la spatola. Aggiungere quindi `1/2` del rimanente miscuglio e mescolare, e infine tutto il miscuglio restante, mescolando sempre molto delicatamente.
- [ ] Versare in una teglia imburrata e infarinata.

## :lucide-heater: Cottura

Cuocere in forno preriscaldato ventilato a `160°C` per `35min`.
