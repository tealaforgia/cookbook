# Pan di Spagna

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `250g [≈5]` | Uovo |
| :fontawesome-regular-square: `65g + stampo` | Farina |
| :fontawesome-regular-square: `65g` | Amido di mais[^1] |
| :fontawesome-regular-square: `125g` | Zucchero |
| :fontawesome-regular-square: `1/2 limone` | Scorza di limone grattugiata |
| :fontawesome-regular-square: `1 cucchiaino` | Essenza di vaniglia |
| :fontawesome-regular-square: `1 pizzico` | Sale |
| :fontawesome-regular-square: `stampo` | Burro |

???- example "Proporzioni"

    **Uovo** (`g`) = `n`

    | Quantità | Ingrediente |
    | :-- | :-- |
    | `n×13÷50` | Farina |
    | `n×13÷50` | Amido di mais |
    | `n÷2` | Zucchero |

---

## :lucide-chef-hat: Preparazione

- [ ] Montare a neve fermissima gli albumi con `1/2` dello zucchero.
- [ ] Montare i tuorli con lo zucchero rimanente, l'essenza di vaniglia, la scorza di limone e un pizzico di sale per `≈15min`.
- [ ] Aggiungere ai tuorli `1/3` di albume, `1/2` di farina + maizena setacciate, `1/3` di albume, la farina + fecola rimanenti e l'albume rimanente, mescolando molto delicatamente di volta in volta con la spatola.
- [ ] Versare in una teglia imburrata e infarinata.

## :lucide-heater: Cottura

Cuocere in forno preriscaldato ventilato a `160°C` per `30min`.

[^1]: Fecola di patate; Amido di frumento
