# Pasta frolla

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `110g [≈2]` | Uovo |
| :fontawesome-regular-square: `500g` | Farina |
| :fontawesome-regular-square: `250g` | Burro |
| :fontawesome-regular-square: `110g` | <span class="opzionale">Zucchero a velo</span>[^1]
| :fontawesome-regular-square: `1 limone` | Scorza di limone grattugiata |
| :fontawesome-regular-square: `1 pizzico` | Sale |

???- example "Proporzioni"

    **Uovo** (`g`) = `n`

    | Quantità | Ingrediente |
    | :-- | :-- |
    | `n×50÷11` | Farina |
    | `n×15÷11` | Burro |
    | `n` | Zucchero a velo |

---

## :lucide-chef-hat: Preparazione

- [ ] Frullare o miscelare con la foglia alla massima velocità la farina con il burro freddo tagliato a pezzetti.
- [ ] Aggiungere la scorza di limone <span class="opzionale">e lo zucchero a velo</span>, quindi amalgamare.
- [ ] Sostituire la foglia con il gancio, quindi aggiungere le uova e impastare.
- [ ] Compattare l'impasto e lasciarlo riposare in frigo per `2-12h`.

## :lucide-heater: Cottura

=== "Crostate; Calzoni; Rustici"

    Cuocere in forno preriscaldato non ventilato a `150-170°C`.

=== "Biscotti; Cotture alla cieca"

    Cuocere in forno preriscaldato non ventilato a `180°C`.


[^1]: Solo nella versione dolce
