# Pomodoro confit

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `500g` | Pomodoro ciliegino[^1] |
| :fontawesome-regular-square: `30g` | Zucchero[^1] |
| :fontawesome-regular-square: | Olio di oliva |
| :fontawesome-regular-square: | Timo |
| :fontawesome-regular-square: | Origano |
| :fontawesome-regular-square: | Sale |
| :fontawesome-regular-square: | Pepe |

---

## :lucide-chef-hat: Preparazione

- [ ] Tagliare i pomodori a metà e metterli in una teglia
- [ ] Condire i pomodori con zucchero e olio, origano, sale e pepe. Mescolare per uniformare il condimento.

## :lucide-heater: Cottura

Cuocere in forno statico a `120°C` per `3h`.

[^1]: Datterino giallo
[^2]: Zucchero di canna Mascobado
