# Pasta con fave e pecorino

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `≈300g` | Fave[^1] |
| :fontawesome-regular-square: `≈300g` | Pasta corta |
| :fontawesome-regular-square: | Pecorino romano |
| :fontawesome-regular-square: | Cipollotto[^2] |
| :fontawesome-regular-square: | Finocchietto selvatico[^3] |
| :fontawesome-regular-square: | Olio di oliva |
| :fontawesome-regular-square: | Sale |
| :fontawesome-regular-square: | Pepe |

---

## :lucide-chef-hat: Preparazione

- [ ] Tritare finemente il cipollotto e farlo soffriggere con l'olio di oliva in una pentola. Mettere a bollire dell'acqua in un altro pentolino.
- [ ] Aggiungere le fave al soffritto e una punta del finocchietto selvatico. Far rosolare per `≈30s` e aggiungere un mestolo di acqua bollente.
- [ ] Far cuociere per `10min`[^4] a fuoco molto basso.
- [ ] Aggiungere sale e pepe, quindi regolare di acqua, che deve appena sommergere le fave.
- [ ] Aggiungere la pasta e cuociere al dente, aggiungendo acqua al bisogno.
- [ ] Aggiungere, a fuoco spento, il pecorino, il finocchietto selvatico, e un po' di pepe.
- [ ] Nel piatto aggiungere un filo di olio di oliva a crudo.

[^1]: Fresche decorticate - In alternativa fave secche decorticate, quantità dimezzata
[^2]: Porro; Cipolla
[^3]: Aneto
[^4]: Se si usano le fave secche, raddoppiare i tempi di cottura
