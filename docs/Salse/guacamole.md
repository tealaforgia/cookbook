# Guacamole

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `1` | Avocado |
| :fontawesome-regular-square: `≈1.5 lime` | Succo di lime[^1] |
| :fontawesome-regular-square: `1/2` | Cipolla |
| :fontawesome-regular-square: `≈1-2` | Pomodoro da insalata |
| :fontawesome-regular-square: | Peperoncino Jalapeño[^2] |
| :fontawesome-regular-square: | Coriandolo |
| :fontawesome-regular-square: | Sale |
| :fontawesome-regular-square: `2 spicchi` | <span class="opzionale">Aglio</span> |

---

## :lucide-chef-hat: Preparazione

- [ ] Tritate molto finemente il coriandolo, <span class="opzionale">l'aglio</span>, la cipolla e il peperoncino, quindi tagliare a cubetti molto piccoli il pomodoro eliminando accuratamente i semi e l'acqua di vegetazione.
- [ ] Aprire l'avocado, versare sopra il succo di lime e schiacciarlo con la forchetta.
- [ ] Regolare di sale ed aggiungere il resto degli ingredienti.

[^1]: Succo di limone
[^2]: Tabasco verde
