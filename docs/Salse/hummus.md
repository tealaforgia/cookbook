# Hummus [حمص بطحينة]

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: `500g` | Ceci lessi[^1] |
| :fontawesome-regular-square: `50-150g` | [Tahina](../Preparazioni-di-base/tahina/#ingredienti){ data-preview } |
| :fontawesome-regular-square: `≈70g` | Olio di oliva |
| :fontawesome-regular-square: `1 spicchio` | Aglio |
| :fontawesome-regular-square: `70g` | Succo di limone |
| :fontawesome-regular-square: | Cumino in polvere |
| :fontawesome-regular-square: `≈30-60g` | Acqua di cottura dei ceci[^2] |
| :fontawesome-regular-square: | Sale |
| :fontawesome-regular-square: | Pepe |
| :fontawesome-regular-square: | Paprika piccante[^3] |
| :fontawesome-regular-square: | Prezzemolo |

---

## :lucide-chef-hat: Preparazione

- [ ] Frullare i ceci con la [tahina](../Preparazioni-di-base/tahina/#preparazione){ data-preview }, il succo di limone, l'olio, l'aglio schiacciato, il sale, il pepe e il cumino.
- [ ] Aggiungere l'acqua di cottura dei ceci e continuare a frullare fino alla consistenza desiderata.
- [ ] Servire condito con prezzemolo, paprika e olio.

[^1]: Ammollo: `48h`; Cottura: `≈2h`
[^2]: Acqua
[^3]: Paprika affumicata; Paprika dolce
