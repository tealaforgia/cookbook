# Salsa Barbecue

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: | Olio di oliva |
| :fontawesome-regular-square: `2 spicchi` | Aglio |
| :fontawesome-regular-square: `1` | Cipolla |
| :fontawesome-regular-square: | Paprika affumicata[^1] |
| :fontawesome-regular-square: | Pepe nero |
| :fontawesome-regular-square: `15g` | Zucchero di canna Mascobado |
| :fontawesome-regular-square: | Aceto di mele[^2] |
| :fontawesome-regular-square: | Tabasco |
| :fontawesome-regular-square: | Salsa Worcester |
| :fontawesome-regular-square: `10g` | Sciroppo d'acero |
| :fontawesome-regular-square: `100-150g` | Ketchup |

---

## :lucide-chef-hat: Preparazione

- [ ] Tritare molto finemente aglio e cipolla e farli rosolare in padella con l'olio.
- [ ] Aggiungere la paprika, il pepe e lo zucchero di canna, quindi far caramellare leggermente lo zucchero.
- [ ] Sfumare con l'aceto di mele e aggiungere il tabasco, la salsa Worcester e lo sciroppo d'acero, quindi aggiungere il ketchup.
- [ ] Far riprendere il bollore e cuocere a fuoco basso finché non raggiunge la consistenza desiderata.

[^1]: Paprika dolce; Paprika piccante  
[^2]: Aceto di vino bianco
