# Alici con patate al forno

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: | Filetti di alici |
| :fontawesome-regular-square: | Patate |
| :fontawesome-regular-square: | Aglio |
| :fontawesome-regular-square: | Prezzemolo |
| :fontawesome-regular-square: | Olio di oliva |
| :fontawesome-regular-square: | Sale |
| :fontawesome-regular-square: | Pepe |

---

## :lucide-chef-hat: Preparazione

- [ ] Tagliare le patate a fettine e cuocierle `≈5min` in acqua bollente
- [ ] Disporre nella teglia, nell'ordine, olio, patate rigorosamente asciutte, alici rigorosamente asciutte, sale grosso, aglio tritato, olio, sale, pepe, prezzemolo tritato.

## :lucide-heater: Cottura

Cuocere in forno preriscaldato ventilato a `180°C` per `≈10-15min`.
