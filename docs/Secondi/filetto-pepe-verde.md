# Filetto al pepe verde

## :lucide-shopping-basket: Ingredienti

| Quantità | Ingrediente |
| :-- | :-- |
| :fontawesome-regular-square: | Filetto[^1] |
| :fontawesome-regular-square: | Pepe verde |
| :fontawesome-regular-square: | Senape di digione |
| :fontawesome-regular-square: | Cognac[^2] |
| :fontawesome-regular-square: | Burro |
| :fontawesome-regular-square: | Panna |
| :fontawesome-regular-square: | Brodo di carne[^3] |
| :fontawesome-regular-square: | Sale |
| :fontawesome-regular-square: | <span class="opzionale">Patate al forno</span>[^4] |

---

## :lucide-chef-hat: Preparazione

- [ ] Salare la carne. <span class="opzionale">Cuocerla a temperatura controllata a `54-60°C`, recuperando il liquido di cottura.</span>
- [ ] Cuocere la carne in padella per farla dorare.
- [ ] Abbassare la teperatura della padella, aggiungere il burro, quindi sfumare col Cognac.
- [ ] Togliere la carne e aggiungere il brodo.
- [ ] Aggiungere il pepe verde, la panna e la senape, far ridurre ad una crema. Nel frattempo tagliare la carne a fettine.
- [ ] Impiattare la carne aggiungendo la crema sopra la carne <span class="opzionale">e le patate al lato</span>.
